1 Halting Problem

1.1 问题

是否存在一个函数 halts(g), 使用对任意函数 g, halts 都能返回:

  1. true, 表示 f 在有限时间内结束
  2. false, 表示 f 不能在有限时间内结束

1.2 证明

定义一个函数 g: 

def g():
    if halts(g):
        loop_forever()

显然 halts 无论返回 true 或 false 都会产生矛盾, 所以 halts 函数不存在.
