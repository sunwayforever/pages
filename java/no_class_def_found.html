1 No Class Def Found

// 2019-10-29 13:20
class ErrorClass {
    static {
        int x = 1 / 0;
    }
}

public class Test {

    public static void main(String argv[]) {
        try {
            new ErrorClass();
        } catch (Throwable e) {
            e.printStackTrace(System.out);
        }
        System.out.println("------------------");
        try {
            new ErrorClass();
        } catch (Throwable e) {
            e.printStackTrace(System.out);
        }
    }
}

java.lang.ExceptionInInitializerError at Test.main(Test.java:12) Caused by: java.lang.ArithmeticException: / by zero at ErrorClass.<clinit>(Test.java:4) … 1 more

java.lang.NoClassDefFoundError: Could not initialize class ErrorClass at Test.main(Test.java:18)

NoClassDefFoundError 与 ClassNotFound 不同, 后者是找不到 class, 前者是能找到 class, 但 LoadClass 时出错了, 例如:

  1. clinit 出错
  2. verify 出错
  3. resolve field 出错
  4. …

如果后续再次 load 这个 class, 则会直接报 NoClassDefFoundError, 例如 art 中的 `ThrowEarlierClassFailure` 方法
