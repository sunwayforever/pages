1 Rust Tips

1.1 build rust executable for android   toolchain android cross_compile rust

  1. rustup target add aarch64-linux-android

  2. cat ~/.cargo/config 

    [target.aarch64-linux-android]
linker = "aarch64-linux-android-gcc"
  3. cargo build –target=aarch64-linux-android
