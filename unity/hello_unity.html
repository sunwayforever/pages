1 Hello Unity

1.1 移动, 视角

  • Alt+left 旋转， middle 移动
  • Q,W,E,R 切换操作模式:平移,按坐标轴和坐标平面移动,旋转,scale, …
  • f 或双击来 focus object
  • 右上角选择 x,y,z 视角
  • 右上角中间选择是否是 perspective 视角, 非 perspective 视角适合用来对齐 object
  • 右上角选 default 来恢复原始布局
  • 按ctrl再移动物体时会考虑 snap
  • object 指定给特定的 layer 后， 可以通过锁定或隐藏通过该 layer 来锁定或隐藏所有包含在该 layer 中的 object
  • align with view (C-S-f)

1.4 component

  • rigidbody 给物体添加重力和摩擦力， 没有 rigidbody component 的 object 是不会因为重力或撞击而运动的
  • rigidbody 添加 constraints 使它只能移动不能旋转
  • rigidbody drag 改变阻力
  • boxcllider 给物体添加碰撞
  • boxcllider.is_trigger trigger 会有碰撞 (onTriggerEnter), 但碰撞后后穿透
  • animation

1.5 script

  • script 通过拖动使代码能引用其它的 object 和 component
  • script 中声明的类成员可以直接在 inspector 中查看和修改
  • 通过 object.transform.position 修改 object 的位置
  • 通过 script 使得 camera 和 player 保持相同的 transform.position, 实现 camera 跟踪 player
  • onStart()
  • onUpdate 中通过 Timer.deltaTime 得到两次 update 的时间， 以便使程序能自适应 frame 的快慢
  • rigidbody.addForce(x,y,z,forcemode)
  • Input.GetKey()
  • onCollisionEnter()
  • script component 也可以传入其它 component, 例如通过 script_component.enabled=false 来停用某个 script component
  • FindObjectOfType<>
  • UnityEngine.SceneManagement
  • Invoke("function",delay_in_secs)
  • GameObject.SetActive
  • UnityEngine.UI
  • PlayPrefs
