1 sed tips

1.1 command pattern

sed -options '[range]command;[range]command' file
sed -options '[range]{command;command}' file

1.2 options

1.2.1 -i: to modify the file directly

  • sed -i 's/abc/xyz/' file

1.2.2 -n: no printing

The "-n" option will not print anything unless an explicit request to print is found.

e.g. the `p` command

1.3 pattern

1.3.1 supported expression

  • [-]
  • ^
  • $
  • .
  • *
  • [^]
  • \{N\}

1.3.2 sed is always greedy

To match `abc xyz`

  • \|.* .*| won't work
  • \|[^ ]* [^ ]*| will do the trick

1.3.3 sed doesn't support `+`

  • \|.+| won't work
  • \|..*| will do the trick

1.4 range

1.4.1 range with line number

  • sed '1d'
  • sed '1,10d'
  • sed '$d'

1.4.2 range with pattern

  • sed '\|pattern|d'
  • sed '\|pattern1|,|pattern2|d'

1.4.3 gnu sed extendtion

1.4.3.1 first~step

sed '0~2d' # will match lines 0,2,4,6

1.4.3.2 addr,+N

sed 'pattern,+10' # will match lines from pattern and the following 10 lines
