1 Semantic Segmentation

1.1 Overview

semantic segmentation 的一个核心问题是如何解决 pooling 导致的 representational bottleneck. 所谓的 bottleneck 是指通过多次 pooling 会丢失浅层的特征, 只保留深层 的特征, 深层特性对图片的分类更有用, 但浅层特征对语义分割更有用.

解决的方法大约有:

  1. 通过 dilated conv2d (而不是 pooling) 增大 receptive field

  2. 不同尺度的特征的融合 (例如 skip connection, spp), 以保留浅层特征.

    关于特征融合, 大部分做法是保证 h,w 相同 (例如通过 resize, crop, upsampling 等), 然后针对 c 做 concat.
