1 UNet

https://arxiv.org/pdf/1505.04597.pdf 2015/05

UNet 是一个主要针对医疗影像 (biomedical image) 语义分割的模型

1.1 Network

UNet 也是一个标准的 encoder-decoder 结构, 但有一些需要注意的地方:

  1. 所有的蓝色的 conv 3x3 都是 valid padding, 导致 encoder 和 decoder 的对应部分尺寸不一致
  2. valid padding 也导致最终输出与 input 尺寸不匹配 (input: 572x572, output 388x388), 所以训练时需要给原始图像加 padding 或者在原始图像向外 crop 出更大的图片, 得到 (572x572). 原论文没有采用 same padding 可以是因为 feature map 加 padding 可以会影响精度? 毕竟凭空加入了一些毫无依据的 0
  3. 灰色的 `copy and crop` 类似 resnet 的 skip connection, 但使用的是针对 channel 的 concat 而不是直接相加. 另外, 因为 concat 的两部分的 shape 不一致, 所以需要把大的 crop 成小的
