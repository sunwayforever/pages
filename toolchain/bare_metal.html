1.1 qemu arm bare metal example

https://balau82.wordpress.com/2010/02/28/hello-world-for-bare-metal-arm-using-qemu/

为了验证 kws 在 mips bare metal 上的情况, 使用 qemu arm bare metal example 来测 试 bare metal 的基本概念

支持 bare metal 的方法:

  1. 针对板子写 start.s, 初始化 sp 等, 然后跳转到 main
  2. linker script 需要把 start.o 放在板子要求的地址上, 同时需要提供一些符号, 例如 newlib 中的 _sbrk 需要 linker script 定义 `end` 以后才能支持 malloc
  3. 使用 nosys.specs 把 newlib 做为 libc, 同时使用 libnosys 给 _fork, _exit 等函 数提供 dummy 实现, 避免 newlib 编译不过

1.4 libgcc, crt0.o 与 nostdlib, nodefaultlibs, nostartfiles

1.4.1 libgcc

GCC provides a low-level runtime library, libgcc.a or libgcc_s.so.1 on some platforms. GCC generates calls to routines in this library automatically, whenever it needs to perform some operation that is too complicated to emit inline code for.

  1. libgcc 是为了弥补平台本身对某些功能的支持问题, 例如 soft-float
  2. gcc 会针对某些代码生成对 libgcc 的隐式的调用, 例如 gcc 会把对 thread_local 的 变量的读写的代码编译成对 libgcc 中 emutls 相关函数的调用, 所以针对 libgcc 的 编译错误经常不那么直观
  3. libgcc 的主要功能包括:
    • emutls 相关
    • automic 相关
    • soft float 相关
    • clear_cache
    • 数学运算, 包括乘除移位等

1.4.2 crt0.o

crt0.o 会提供一个 _start 符号, 它的代码差不多是这样的: 

/* ida 反编译的结果 */
void __fastcall start() {
    int v0;  // r0@4

    stack_init();
    memset();
    if (atexit) atexit(_libc_fini_array);
    _libc_init_array();
    v0 = main(0, 0);
    exit(v0);
}

其中 _libc_init_array 会调用 .init_array 中的函数指针 

/* ida 反编译的结果 */
int __fastcall _libc_init_array()
{
    int result; // r0@3
    int v5; // r4@4
    unsigned int v6; // r5@4
    int (__fastcall *v7)(_DWORD); // t1@5

    result = init_proc();
    if ( 0xC >> 2 )
    {
        v5 = 0;

        /*
         * ~/download@sunway-t14> readelf -a ./a.out |grep __init_array_start
         *           100: 000186cc     0 NOTYPE  LOCAL  DEFAULT    7 __init_array_start
         *           
         */

        v6 = 0x186C8u;
        do
        {
            /*
             * * 0x186c8+4 是 __init_array_start 的第一个函数指针的地址            
             */
            v7 = *(int (__fastcall **)(_DWORD))(v6 + 4);
            v6 += 4;
            ++v5;
            /* 调用这个函数 */
            result = v7(result);
        }
        while ( 0xC >> 2 != v5 );
    }
    return result;
}
