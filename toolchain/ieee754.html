1 IEEE754

float 的格式: 

<- 1-> <-   8  -> <-  23  -> 
----------------------------
<sign> <exponent> <fraction>

以 11.01*2^(-4) 为例:

首先需要转换为 1.x * 2^(n) 的形式 1.101*2(-4)

sign 为 0
exponent 为 -4+127, 其中 127 是 exponent 的 bias
fraction 为 b10100000000000000000000 (而不是 b110100..., 最开头的 1 编码时需要省略掉, 也不是 b000...101)

double 格式: 

<- 1-> <-  12  -> <-  51  -> 
----------------------------
<sign> <exponent> <fraction>

exponent bias 为 1023

1.1 BF16

另外, 存要非标准的 bf16 格式, 它与 float 的唯一区别是 fraction 从 23 位变为 7 位, 导致它精度会下降, 但它与 float 转换非常容易: bf16 末尾添 16 个 0 就可以转换为 float, float 取高 16 位就可以转换为 bf16
